# Embeddable Items in Posts

Posts can include data visualizations, media from Core Data, places from Core Data, and external AV.

## Data Visualizations

## Place

Individual [places can be embedded](https://community.performantsoftware.com/t/how-do-i-embed-a-place-in-a-post/54) from Core Data. Unlike the data visualizations, these are not static, so updates to a place record in Core Data will be reflected here.
